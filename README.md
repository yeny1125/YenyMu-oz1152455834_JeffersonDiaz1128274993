# SPA CRUD

Proyecto desarrollado en Python para la gestión de productos de un spa mediante operaciones CRUD.

## Funcionalidades

- Crear productos
- Consultar productos
- Actualizar productos
- Eliminar productos

## Tecnologías

- Python 3.11
- JSON
- unittest
- GitHub Actions
- SonarQube

## Ejecución

```bash
python main.py
```

## Ejecutar pruebas

```bash
python -m unittest -v
```

## Cobertura

```bash
coverage run -m unittest
coverage report
coverage xml
```
